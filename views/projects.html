Amanda Lynn Dashboard

I made a locally-running app for my (non-technical) Mom, who needed an app that showed her product for her small business. It tracks her revenue vs. expenses, stock, sales, and more! I update it regularly per her requests. It runs on a Vue frontend, Django for backend, SQLite for storage, and docker to build and run the app. Since she is not a tech expert, docker was able to help her start and run the new app updates and changes without using the command line!

Billiards Bonanza

An 8-Ball game with a bunch of twists and funny add-ons. I made it in Unity using C#, and did all of the assets myself. This was a great experience in learning the syntax of C#, and learning more about programming languages and Unity!

Discussion Board

Here is a live discussion board and chat app that was made directly in Java, using live, hand-rolled web sockets, in order to live chat with other people. There are custom options for priveleges and chatting lobbies!

Resume2Web
A Hackathon app that uses Vite, React/TypeScript, and Python/Flask. It takes a resume and converts it to a website, with custom styling. This hackathon was my first hackathon, and helped me learn more about full-stack, and learn new skills in a quick manner!
